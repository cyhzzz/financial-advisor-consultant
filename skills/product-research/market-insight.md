# Skill: 市场观点与晨会准备 (Market Insight & Morning Meeting)

## Description
每日市场分析与观点输出，支持晨会材料、客户市场点评、周报月报等多场景内容生成。覆盖 A 股、基金、宏观、行业等多维度。

## Trigger Conditions
- 每日开盘前准备晨会材料
- 盘后发布市场点评（微信公众号/朋友圈）
- 周报/月报/季报撰写
- 重大市场事件解读
- 客户群日常内容维护

## Workflow

### 步骤 1: 盘前数据扫描 (8:00-9:00)

**全球市场概览**:
```
□ 美股收盘（道指/标普500/纳指）
□ 中概股表现（金龙指数）
□ 欧洲市场（德/法/英）
□ 亚太早盘（日/韩/港）
□ 期货盘前（标普500期货、A50期货）
```

**关键资产价格**:
```
□ 美元指数 (DXY)
□ 人民币汇率 (离岸/在岸)
□ 美债收益率 (2年/10年)
□ 原油 (WTI/布伦特)
□ 黄金 (伦敦金/COMEX)
□ 铜价 (LME)
```

**重要事件提醒**:
```
□ 今日经济数据发布（中国/美国/欧洲 PMI、CPI、就业等）
□ 央行政策动态（利率决策、讲话）
□ 重要会议（政治局会议、美联储议息等）
□ 个股事件（财报发布、大额解禁、分红派息）
```

### 步骤 2: A 股市场分析

**技术面扫描**:
```
□ 主要指数（上证/深证/创业板/科创50/北证50）
  - 收盘价、涨跌幅、成交额
  - 技术形态（均线排列、支撑压力位）
  - 成交量变化（对比5日/20日均量）

□ 市场情绪指标
  - 涨跌家数比
  - 涨跌停家数
  - 北向资金流向（全天/盘中）
  - 融资余额变化
```

**资金面分析**:
```
□ 成交量能
  - 两市总成交额（对比近期均值）
  - 量能趋势（放量/缩量）

□ 资金流向
  - 北向资金（沪股通/深股通）
  - 主力资金（大单/超大单流向）
  - 融资买入/偿还

□ 增量资金
  - 新发基金规模
  - ETF 份额变化
  - 两融余额
```

**板块与行业**:
```
□ 领涨板块（涨幅前10）
  - 上涨逻辑（政策/业绩/事件驱动）
  - 持续性判断

□ 领跌板块（跌幅前10）
  - 下跌原因（业绩暴雷/政策打压/情绪踩踏）
  - 风险是否释放

□ 主题概念热度
  - 涨停个股分布
  - 连板高度
  - 龙头股表现

□ 行业景气度跟踪
  - 周期品（钢铁/煤炭/化工/有色）价格
  - 消费品（白酒/家电/医药）动销
  - 科技品（半导体/新能源）订单
```

### 步骤 3: 基金市场分析

**新发基金**:
```
□ 本周新发（偏股/偏债/指数/REITs）
□ 明星基金经理新发产品
□ 募集规模与募集期
□ 渠道认购热度
```

**存量基金动态**:
```
□ 热门赛道基金份额变化（半导体/医药/新能源）
□ 货币基金收益率排名
□ 短债基金业绩表现
□ FOF/养老目标基金表现
```

**基金策略观点**:
```
□ 明星基金经理近期观点（季报/路演/采访）
□ 头部机构年度策略展望
□ 基金仓位变化（股票仓位/行业配置）
```

### 步骤 4: 宏观与政策解读

**国内宏观**:
```
□ 经济数据解读（PMI/GDP/CPI/PPI/社融）
□ 货币政策动态（MLF/LPR/逆回购）
□ 财政政策动向（专项债/减税降费）
□ 监管政策变化（行业监管/资本市场改革）
```

**国际宏观**:
```
□ 美联储政策（利率决议/缩表/点阵图）
□ 全球经济景气度（美/欧/日 PMI）
□ 地缘政治风险（冲突/制裁/大选）
□ 全球资本流动（新兴市场资金流向）
```

### 步骤 5: 观点形成与策略建议

**市场判断**:
```
□ 短期判断（1周内）：震荡/上行/下行
□ 中期判断（1-3月）：乐观/中性/谨慎
□ 长期判断（1年+）：趋势方向
```

**配置建议**:
```
□ 权益仓位建议（激进/中性/防御）
□ 行业配置建议（超配/标配/低配）
□ 风格配置建议（价值/成长/均衡）
□ 基金类型建议（主动/被动/固收+）
```

**关注方向**:
```
□ 推荐关注板块（列明逻辑）
□ 风险提示板块（列明风险）
□ 潜在机会跟踪（催化剂+时点）
```

## Output Templates

### 1. 晨会 PPT (8-10 页)

**结构**:
```
第1页: 封面（日期、主题）
第2页: 全球市场概览（隔夜美股/汇率/大宗商品）
第3页: A股盘面回顾（涨跌/成交/北向/情绪指标）
第4页: 板块热点（领涨领跌/资金流向）
第5页: 基金市场动态（新发/份额变化/观点）
第6页: 宏观政策要点（国内/国际）
第7页: 今日关注（财经日历/事件提醒）
第8页: 投顾观点（市场判断/配置建议）
第9页: 重点产品推荐（1-2只基金）
第10页: Q&A / 结束
```

### 2. 每日市场点评（微信公众号/朋友圈）

**模板 A: 详细版 (500-800 字)**
```markdown
📊 {日期} 市场点评 | {标题}

【盘面回顾】
今日两市{上涨/下跌}，沪指{涨跌}X.XX%，深成指{涨跌}X.XX%，创业板指{涨跌}X.XX%。两市成交{XXX}亿元，较昨日{放量/缩量}X%。

北向资金{净流入/净流出}XX亿元，其中沪股通{流入/流出}XX亿，深股通{流入/流出}XX亿。

【板块表现】
领涨板块：{板块1}(+X%)、{板块2}(+X%)、{板块3}(+X%)
领跌板块：{板块1}(-X%)、{板块2}(-X%)、{板块3}(-X%)

【热点解读】
{对领涨板块或重要事件的解读，2-3段}

【投顾观点】
{市场判断 + 配置建议，2-3段}

【风险提示】
市场有风险，投资需谨慎。本观点仅供参考，不构成投资建议。

---
关注【{投顾姓名}】，获取更多专业市场观点 👆
```
