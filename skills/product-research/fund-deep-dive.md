# Skill: 基金深度研究 (Fund Deep Dive)

## Description
对单只基金产品进行全方位深度研究，覆盖基金基本信息、历史业绩、持仓分析、基金经理、风险评估等维度，形成专业的研究报告。

## Trigger Conditions
- 重点池基金定期跟踪
- 新客户推荐前的深度研究
- 持仓基金出现异常时的排查
- 撰写专题研究报告

## Workflow

### 步骤 1: 基本信息梳理

**基金概况**:
- 基金全称、简称、代码
- 基金类型（股票型/混合型/债券型/指数型/QDII/FOF等）
- 成立日期、成立规模
- 最新规模（合计/份额）
- 基金管理人、托管人

**运作信息**:
- 投资目标与范围
- 业绩比较基准
- 分红政策
- 申赎规则（开放时间、最低申购额、赎回费结构）

**费率结构**:
- 管理费、托管费、销售服务费
- 申购费（前端/后端，折扣情况）
- 赎回费（持有期阶梯费率）

### 步骤 5: 风险评估

**信用风险**:
```
□ 债券持仓信用评级分布
□ 重仓债券发行人信用状况
□ 行业集中度风险
□ 地域集中度风险
```

**流动性风险**:
```
□ 组合平均流动性评分
□ 重仓股日均成交额
□ 基金规模与流动性匹配度
□ 申赎压力测试
```

**市场风险**:
```
□ Beta 系数（系统性风险暴露）
□ 行业偏离度（vs 基准）
□ 风格偏离度（vs 基准）
□ 极端市场情景压力测试
```

**操作风险**:
```
□ 基金公司运营稳定性
□ 基金经理变更风险
□ 合规记录
```

```
分析:

```
