# Skill: 产品推介材料制作 (Product Collateral Creation)

## Description
为各类金融产品（基金、理财、保险、信托等）制作专业的推介材料，包括一页纸精华版、详细推介PPT、FAQ文档、营销话术卡等，确保合规、专业、有说服力。

## Trigger Conditions
- 新产品上线需要制作全套推介材料
- 重点产品销售期需要定制化材料
- 针对特定客户群体（如高净值客户）定制材料
- 参加路演、客户沙龙需要演讲材料

## Workflow

### 步骤 1: 产品信息梳理

**基础信息收集**:
```
□ 产品名称、代码、类型
□ 发行机构、托管机构
□ 成立日期、产品期限
□ 投资范围与比例限制
□ 业绩比较基准
```

**策略与特点**:
```
□ 投资策略描述（定性）
□ 量化模型/选股标准（如有）
□ 风险控制措施
□ 与同类产品的主要差异
□ 核心竞争优势
```

**费率与收益**:
```
□ 认购/申购费率结构
□ 管理费率、托管费率
□ 业绩报酬（如有）
□ 历史业绩（各时间段）
□ 分红政策
```

**风险提示**:
```
□ 产品风险等级
□ 主要风险因素
□ 最不利投资情形
□ 流动性风险
□ 适当性匹配要求
```

### 步骤 2: 卖点提炼与定位

**核心卖点（3-5个）**:
```
1. {卖点1}: {一句话描述} → 支撑论据
2. {卖点2}: {一句话描述} → 支撑论据
3. {卖点3}: {一句话描述} → 支撑论据
...
```

**目标客群画像**:
```
□ 年龄段: {范围}
□ 风险承受能力: {R1-R5}
□ 投资经验: {新手/有一定经验/经验丰富}
□ 资产规模: {范围}
□ 投资期限: {短期/中期/长期}
□ 投资目标: {保值增值/养老/教育/其他}
□ 不适合人群: {明确列出}
```

**竞争定位**:
```
□ 直接竞品: {列出2-3个}
□ 竞争优势: {与竞品相比的优势}
□ 竞争劣势: {客观存在的劣势}
□ 差异化策略: {如何避开竞争劣势，突出优势}
```

### 步骤 3: 材料制作

**设计建议**:
- 使用公司统一的VI设计
- 重点数据用颜色突出
- 控制在一页A4纸内（正反面也可）
- 便于打印和微信转发

#### 3.2 详细推介 PPT（10-15页）

**用途**: 路演、客户沙龙、深度介绍

**页面结构**:
```
第1页:  封面（产品名称、公司Logo、日期）
第2页:  目录
第3页:  产品定位（一句话+核心亮点）
第4页:  市场背景（为什么需要这个产品）
第5页:  产品策略（投资范围、策略、流程）
第6页:  团队介绍（基金经理、投研团队）
第7页:  历史业绩（收益、风险、排名）
第8页:  竞品对比（与主要竞品的优劣势）
第9页:  适合人群（谁适合/谁不适合）
第10页: 费率结构
第11页: 风险提示
第12页: Q&A
第13页: 联系方式/结束页
```

**设计原则**:
- 每页一个核心信息点
- 多用图表，少用文字
- 数据可视化
- 统一的视觉风格
- 便于演讲者讲解的备注栏

## 常见客户疑问及回应

### Q1: "这个产品和XX产品有什么区别？"
**核心差异**: {一句话总结核心区别}
**详细对比**:
- {维度1}: 本产品{特点} vs XX产品{特点}
- {维度2}: 本产品{特点} vs XX产品{特点}
- {维度3}: 本产品{特点} vs XX产品{特点}
**选择建议**: {什么情况下选本产品，什么情况下选XX产品}

### Q2: "现在市场这么差，买这个产品安全吗？"
**回应要点**:
1. 首先理解客户对"安全"的定义（保本？不亏？波动小？）
2. 说明风险和收益的关系
3. 介绍本产品的风控措施
4. 展示历史在市场不好时的表现
5. 强调长期投资和资产配置的重要性

**话术示例**:
"我理解您的担心。确实，任何投资都有风险，没有人能保证一定赚钱。

但我们可以从几个角度来看这个问题：

第一，风险和收益是成正比的。如果您完全不能承受任何波动，那可能更适合存款或货币基金。但如果您能接受一定的波动，长期来看，权益类产品是有机会获得更高收益的。

第二，看这款产品的风险控制。{介绍具体风控措施}。

第三，看历史表现。{介绍历史上市场不好时的表现}。

第四，也是最重要的，要看整体配置。不建议把所有钱都放在一个产品上，分散投资可以降低风险。

所以我的建议是：如果您担心市场，可以考虑{分批买入/定投/先少买一点试试/配置一部分}。您觉得这个建议怎么样？"

### Q3: "这个产品的收益能到多少？"
**合规回应**:
"根据监管规定，我们不能对产品的未来收益作出任何承诺或保证。过往业绩不代表未来表现。

但我可以和您分享一下历史数据作为参考：
- 成立以来年化收益：{X}%
- 近1年收益：{X}%
- 近3年年化：{X}%
- 在同类产品中排名：前{X}%

当然，这些数据只是历史表现，未来的收益会受到市场环境、基金经理操作等多种因素影响，可能高于也可能低于历史水平。

投资有风险，您需要根据自己的风险承受能力来决定是否投资以及投资多少。"

### Q4: "我要是买了之后想赎回了怎么办？"
**回应要点**:
1. 说明赎回的流程
2. 说明赎回到账时间
3. 说明赎回费用（与持有期的关系）
4. 提醒短期赎回可能不划算
5. 建议的持有期

**话术示例**:
"关于赎回，是这样的：

第一，赎回很方便。您可以通过{渠道}随时提交赎回申请。

第二，到账时间。一般情况下，{T+X}个工作日可以到账。

第三，赎回费用。这个和您持有的时间有关：
- 持有不满{X}天，赎回费是{X}%
- 持有满{X}天但不满{X}天，赎回费是{X}%
- 持有满{X}天，免赎回费

所以这里要提醒您，如果买了很短时间就赎回，赎回费会比较高，可能不划算。

第四，建议的持有期。考虑到{产品特点/投资策略}，建议至少持有{X}个月以上。

您看还有什么问题吗？"

## 促成交易话术

### 试探性成交
"{客户称呼}，通过刚才的介绍，您觉得这个产品符合您的需求吗？"
"如果您觉得还可以，我建议您可以{先买一点试试/作为配置的一部分/定投}，您看怎么样？"

### 假设成交
"{客户称呼}，那我就帮您申购{X}万元，您看可以吗？"
"好的，那我用您尾号{XXXX}的银行卡帮您操作，您确认一下。"

### 紧迫感促成
"{客户称呼}，这个产品目前额度比较紧张/优惠费率截止到{X月X日}，建议您今天先确定下来，我帮您预留额度。"

### 二选一法
"{客户称呼}，您是打算先投{X}万试试，还是直接配置{X}万？"

## 禁忌话术（绝对不能说）

❌ "保证赚钱" / "肯定能到{X}%" / "绝对不会亏"
❌ "这个产品没有任何风险" / "绝对安全"
❌ "其他产品都不好，就买这个" / "那个产品肯定不行"
❌ "我自己也买了"（如果实际没买或不能证明）
❌ "错过这次机会就没有了" / "今天不买以后就买不到了"
❌ "你不懂，听我的就对了" / "这个太复杂，说了你也不懂"
❌ 透露其他客户的持仓信息或收益情况
❌ 对竞品进行恶意贬低或传播未经证实的负面信息

## 合规注意事项

⚠️ 所有话术必须基于事实，不得夸大或虚假宣传
⚠️ 不得承诺收益或保证本金安全
⚠️ 必须充分揭示风险，确保客户理解
⚠️ 适当性匹配，不得向风险不匹配的客户推介
⚠️ 保留沟通记录，以备合规检查
