# Skill: A股市场分析 (A-Share Market Analysis)

## Description
对A股市场进行全方位分析，包括大盘走势、技术面、资金面、情绪面等多维度解读，为投资决策提供市场观点支撑。

## Trigger Conditions
- 每日开盘前/收盘后市场分析
- 市场出现重大波动时
- 客户询问市场观点
- 定期市场回顾报告

## Workflow

### 步骤 1: 大盘指数分析

**主要指数跟踪**:
```
□ 上证指数（大盘蓝筹，市场情绪标杆）
□ 深证成指（深市整体表现）
□ 创业板指（成长股、科技股）
□ 科创50（科创板硬科技）
□ 北证50（专精特新小巨人）
□ 沪深300（A股核心资产）
□ 中证500（中小盘成长）
□ 中证1000（小盘股）
```

**指数分析维度**:
```
1. 价格走势
   - 开盘价、最高价、最低价、收盘价
   - 涨跌点数、涨跌幅
   - 日内走势（高开低走、低开高走、震荡等）

2. 成交量能
   - 成交量（相比5日/20日均量）
   - 成交额
   - 量能趋势（放量/缩量）

3. 技术形态
   - 关键点位（支撑位、压力位）
   - 均线系统（5日、10日、20日、60日、120日、250日）
   - MACD、KDJ、RSI等指标
   - 形态识别（头肩顶/底、双顶/底、三角形等）

4. 市场广度
   - 上涨家数 vs 下跌家数
   - 涨停家数 vs 跌停家数
   - 涨幅>5%家数 vs 跌幅>5%家数
```

### 步骤 2: 资金面分析

**北向资金（外资）**:
```
□ 全天净流入/流出金额
□ 沪股通净流入/流出
□ 深股通净流入/流出
□ 近5日、近20日累计流向
□ 重点买入/卖出个股
□ 与大盘走势的相关性分析
```

**两融资金（杠杆资金）**:
```
□ 融资余额变化
□ 融资买入额/偿还额
□ 融资余额占流通市值比例
□ 融券余额变化
□ 融资融券余额差值
```

**主力资金**:
```
□ 大单净流入/流出（通常指>100万）
□ 超大单净流入/流出（通常指>500万）
□ 主力净流入行业分布
□ 主力净流入个股排行
```

**ETF资金流向**:
```
□ 股票型ETF份额变化
□ 宽基ETF（沪深300、中证500等）资金流向
□ 行业ETF资金流向
□ 跨境ETF资金流向
```

### 步骤 3: 板块与行业分析

**领涨板块分析**:
```
□ 涨幅前10板块
□ 领涨原因分析（政策/业绩/事件驱动）
□ 板块内个股表现（涨停家数、涨幅分布）
□ 持续性判断（技术形态、量能配合）
□ 相关产业链联动情况
```

**领跌板块分析**:
```
□ 跌幅前10板块
□ 下跌原因分析（政策利空/业绩暴雷/黑天鹅）
□ 风险是否释放充分
□ 抄底时机判断（估值水平、技术支撑）
```

**行业轮动分析**:
```
□ 当前市场风格（价值/成长、大盘/小盘、周期/消费/科技）
□ 行业轮动节奏（启动→加速→高潮→退潮）
□ 资金在不同行业间的流动
□ 下一阶段可能轮动的方向预判
```

**主题投资分析**:
```
□ 当前热点主题（AI、新能源、机器人、国企改革等）
□ 主题的催化因素（政策、技术突破、事件）
□ 主题的生命周期（萌芽→成长→成熟→衰退）
□ 主题相关标的梳理
□ 主题投资的节奏把握
```

### 步骤 4: 市场情绪分析

**量化情绪指标**:
```
□ 恐慌贪婪指数（0-100）
□ VIX恐慌指数（美股，作为参考）
□ 换手率（全市场、主板、创业板对比）
□ 融资买入占比
□ 新增投资者数量
□ 基金发行规模（冰点/火爆）
□ 上市公司回购/增持情况
```

**情绪观察维度**:
```
□ 媒体氛围（乐观/悲观/中性）
□ 专家观点（一致预期/分歧）
□ 散户情绪（贪婪/恐慌/麻木）
□ 机构行为（加仓/减仓/调仓）
□ 北向资金态度（持续流入/流出/波动）
```

**情绪周期判断**:
```
□ 当前所处阶段（绝望→希望→乐观→狂热→犹豫→恐慌）
□ 情绪转折点信号
□ 情绪与指数的背离（情绪指标与大盘走势不一致）
□ 极端情绪的应对策略
```

### 步骤 5: 政策与宏观分析

**货币政策**:
```
□ 央行公开市场操作（MLF、逆回购等）
□ 利率水平（LPR、MLF利率）
□ 存款准备金率
□ 市场流动性状况（SHIBOR、DR007等）
□ 货币政策基调（宽松/稳健/收紧）
```

**财政政策**:
```
□ 财政政策基调（积极/稳健/紧缩）
□ 专项债发行进度
□ 减税降费政策
□ 财政支出重点方向
□ 地方债务状况
```

**产业政策**:
```
□ 重点支持行业（新能源、半导体、AI等）
□ 监管政策变化（互联网、教育、医疗等）
□ 区域发展政策（京津冀、长三角、粤港澳等）
□ 对外开放政策（自贸区、外资准入等）
```

**宏观经济数据**:
```
□ GDP增速（季度）
□ 工业增加值
□ 固定资产投资
□ 社会消费品零售总额
□ 进出口数据
□ CPI/PPI（通胀水平）
□ PMI（制造业景气度）
□ 就业数据（失业率）
```

**国际环境**:
```
□ 美联储货币政策
□ 美元指数/人民币汇率
□ 地缘政治风险
□ 全球贸易形势
□ 大宗商品价格（原油、黄金、铜等）
```
