# Skill: 客户画像与需求分析 (Client Profiling & Needs Analysis)

## Description
通过系统化信息采集与分析，构建完整的客户画像，精准识别客户需求、风险偏好和投资目标，为个性化服务方案奠定基础。

## Trigger Conditions
- 新客户首次接触
- 年度/半年度客户回访
- 客户发生重大人生变化（结婚、生育、退休等）
- 投资目标调整需求

## Workflow

### 步骤 1: 基础信息采集

**人口统计信息**:
- 姓名、性别、年龄
- 婚姻状况、子女情况
- 居住地、职业、行业
- 教育背景

**财务状况**:
- 年收入（主动收入 + 被动收入）
- 总资产（流动性资产 + 投资性资产 + 自用资产）
- 负债情况（房贷/车贷/信用贷）
- 净资产 = 总资产 - 总负债
- 可投资资产 = 流动性资产 + 部分投资性资产

**现金流分析**:
- 月度固定支出
- 年度大额支出计划
- 应急资金需求（3-6 个月支出）

### 步骤 2: 风险评估 (Risk Profiling)

**定量评估 - 风险测评问卷** (15-20 题):

```
维度1: 投资经验与知识
Q1: 您的投资经验?
    A. 无经验 (1分)  B. <1年 (2分)  C. 1-3年 (3分)  D. 3-5年 (4分)  E. >5年 (5分)

Q2: 对以下投资品种的了解程度?
    [股票] [债券] [基金] [衍生品] [另类投资]
    每项: 不了解(1分)  略知(2分)  熟悉(3分)  精通(4分)

维度2: 风险承受能力
Q3: 您可接受的最大投资亏损比例?
    A. 不能亏损 (1分)  B. <5% (2分)  C. 5-15% (3分)  D. 15-30% (4分)  E. >30% (5分)

Q4: 如果投资亏损20%，您会?
    A. 立即全部卖出 (1分)  B. 卖出一部分 (2分)  C. 持有观望 (3分)  D. 逢低加仓 (4分)  E. 大幅加仓 (5分)

维度3: 投资目标与期限
Q5: 您的投资期限?
    A. <1年 (1分)  B. 1-3年 (2分)  C. 3-5年 (3分)  D. 5-10年 (4分)  E. >10年 (5分)

Q6: 您的主要投资目标?
    A. 保本保息 (1分)  B. 稳定收益 (2分)  C. 稳健增长 (3分)  D. 长期增值 (4分)  E. 追求高回报 (5分)

[其他问题...]
```

**评分与等级对应**:

| 分数区间 | 风险等级 | 等级描述 | 适配权益仓位 |
|----------|----------|----------|--------------|
| 15-25 | R1 | 保守型 | 0-10% |
| 26-40 | R2 | 稳健型 | 10-30% |
| 41-55 | R3 | 平衡型 | 30-50% |
| 56-70 | R4 | 成长型 | 50-70% |
| 71-85 | R5 | 进取型 | 70-100% |

**定性评估 - 深度访谈**:
```
必问问题:
1. 您过去最成功的一次投资是什么？收益多少？为什么成功？
2. 您过去最失败的一次投资是什么？亏损多少？当时感受如何？
3. 如果明天股市下跌20%，您会是什么反应？具体会做什么？
4. 您的这笔钱，最长可以多久不赎回？如果急用怎么办？
5. 您对"高风险高收益"这句话怎么理解？能接受吗？

观察要点:
- 客户谈论投资时的情绪（兴奋/恐惧/理性）
- 对过去盈亏的解释（归因于运气/能力/市场）
- 对风险词汇的反应
- 决策风格（冲动/犹豫/果断）
```

### 步骤 3: 投资目标分析 (Goal-Based Planning)

**目标分解框架**:
```
客户总目标: 财富保值增值
    │
    ├── 短期目标 (1年内)
    │   ├── 应急资金储备: XX万
    │   ├── 年度大额支出: XX万（如：保费、学费）
    │   └── 短期储蓄目标: XX万（如：购车首付）
    │
    ├── 中期目标 (1-5年)
    │   ├── 购房首付: XX万
    │   ├── 子女教育金: XX万
    │   ├── 创业/投资资金: XX万
    │   └── 换车/装修: XX万
    │
    └── 长期目标 (5年以上)
        ├── 退休养老金: XX万（每月XX元×20年）
        ├── 财富传承: XX万
        └── 慈善/公益: XX万
```

**目标可行性检验**:
```
公式: 未来值 = 现值×(1+收益率)^年数 + 年投入×[(1+收益率)^年数-1]/收益率

检验项:
1. 现有可投资资产能否支撑目标？
2. 未来储蓄率是否足够？
3. 目标收益率是否现实（不过高）？
4. 时间跨度是否合理？
5. 多个目标之间是否存在冲突？

调整策略:
- 优先级排序（必须/想要/愿望）
- 延长时间
- 提高储蓄率
- 降低目标金额
- 调整收益率预期
```
