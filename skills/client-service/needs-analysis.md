# Skill: 需求深度分析 (Needs Analysis)

## Description
通过系统化的需求分析框架，深入了解客户的财务目标、风险偏好、投资期限、流动性需求等，为制定个性化的资产配置方案提供基础。

## Trigger Conditions
- 新客户首次KYC
- 客户重大人生变化（结婚、生子、购房、退休等）
- 定期年度/半年度需求回顾
- 投资目标达成度评估

## Workflow

### 步骤 1: 财务目标梳理

**目标分类框架**:

```
按时间维度:
├── 短期目标（1年内）
│   ├── 应急资金储备
│   ├── 年度大额支出（保险、税费等）
│   └── 短期储蓄目标（购车首付、旅行等）
├── 中期目标（1-5年）
│   ├── 购房首付
│   ├── 子女教育金（学前、小学、中学）
│   ├── 创业/投资资金
│   └── 大额消费（换车、装修等）
└── 长期目标（5年以上）
    ├── 退休养老金
    ├── 子女高等教育（大学、研究生）
    ├── 财富传承
    └── 慈善/公益
```

**目标量化模板**:

| 目标名称 | 目标金额 | 目标时间 | 当前进度 | 资金缺口 | 优先级 | 可行性 |
|----------|----------|----------|----------|----------|--------|--------|
| {目标1} | XX万 | X年后 | XX万 | XX万 | P{0-2} | {可行/需调整/不可行} |
| {目标2} | XX万 | X年后 | XX万 | XX万 | P{0-2} | {可行/需调整/不可行} |

**可行性检验公式**:

```
未来值(FV) = 现值(PV) × (1 + r)^n + 年投入(PMT) × [(1 + r)^n - 1] / r

其中:
- PV: 当前可投资资产
- r: 预期年化收益率
- n: 投资年限
- PMT: 每年新增投资

可行性判断:
- 如果 FV ≥ 目标金额: 可行 ✓
- 如果 FV < 目标金额但差距<20%: 需调整 ⚠️
- 如果 FV < 目标金额且差距≥20%: 不可行 ❌
```

### 步骤 2: 风险偏好评估

**定量评估 - 风险测评问卷**:

```
【风险承受能力测评】

1. 您的年龄是？
   A. 25岁以下 (5分)  B. 26-35岁 (4分)  C. 36-45岁 (3分)  D. 46-55岁 (2分)  E. 56岁以上 (1分)

2. 您的投资经验？
   A. 无经验 (1分)  B. 少于1年 (2分)  C. 1-3年 (3分)  D. 3-5年 (4分)  E. 5年以上 (5分)

3. 您的投资知识水平？
   A. 完全不了解 (1分)  B. 略知一二 (2分)  C. 有一定了解 (3分)  D. 比较熟悉 (4分)  E. 非常专业 (5分)

4. 您可接受的最大投资亏损比例？
   A. 不能亏损 (1分)  B. 5%以内 (2分)  C. 5-15% (3分)  D. 15-30% (4分)  E. 30%以上 (5分)

5. 如果投资亏损20%，您会？
   A. 立即全部卖出 (1分)  B. 卖出一部分 (2分)  C. 持有观望 (3分)  D. 逢低补仓 (4分)  E. 大幅加仓 (5分)

6. 您的投资期限？
   A. 1年以内 (1分)  B. 1-3年 (2分)  C. 3-5年 (3分)  D. 5-10年 (4分)  E. 10年以上 (5分)

7. 您的投资目标是？
   A. 保本保息 (1分)  B. 稳定收益 (2分)  C. 稳健增长 (3分)  D. 长期增值 (4分)  E. 追求高回报 (5分)

8. 您的年收入中，可用于投资的资金比例？
   A. 10%以内 (1分)  B. 10-20% (2分)  C. 20-30% (3分)  D. 30-50% (4分)  E. 50%以上 (5分)

9. 您目前的家庭负担情况是？
   A. 未婚，无负担 (5分)  B. 已婚，无子女 (4分)  C. 有子女，父母无需赡养 (3分)  D. 有子女，需赡养父母 (2分)  E. 单亲/特殊家庭情况 (1分)

10. 如果市场出现大幅波动（如±20%），您的情绪反应是？
    A. 非常焦虑，无法入睡 (1分)  B. 比较担心，会影响工作生活 (2分)  C. 略有担心，但生活如常 (3分)  D. 基本不担心，相信长期价值 (4分)  E. 反而兴奋，觉得是机会 (5分)

【评分标准】
- 总分 10-20分：R1 保守型
- 总分 21-30分：R2 稳健型
- 总分 31-40分：R3 平衡型
- 总分 41-45分：R4 成长型
- 总分 46-50分：R5 进取型
```

**定性评估 - 深度访谈**:

```
【投资行为观察】

1. 历史投资经历
   - 最早的投资经历是什么？结果如何？
   - 印象最深刻的一次投资（成功或失败）是什么？
   - 从中学到了什么？

2. 投资决策风格
   - 投资决策前会做哪些准备？
   - 更相信数据/分析还是直觉/感觉？
   - 喜欢独立决策还是参考他人意见？

3. 风险态度
   - 如何看待投资中的亏损？
   - 能否接受"今天赚明天亏"的波动？
   - 投资亏损时会有什么反应？

4. 投资目标认知
   - 对投资收益的期望是多少？
   - 这个期望是基于什么得出的？
   - 是否理解"高收益=高风险"的关系？

【行为金融学测试】

场景1：锚定效应测试
"您第一次看到某基金净值是1.5元，现在跌到1.2元，您觉得便宜吗？如果第一次看到就是1.2元呢？"

场景2：损失厌恶测试
"有两个选择：A. 确定赚5000元；B. 50%概率赚12000元，50%概率赚0元。您选哪个？"
"如果变成：A. 确定亏5000元；B. 50%概率亏12000元，50%概率亏0元。您选哪个？"

场景3：心理账户测试
"您有1万元在股市投资，1万元在银行存款。股市赚了2000元，银行利息200元。您更愿意花哪笔钱去度假？"

场景4：羊群效应测试
"如果身边的亲戚朋友都在买某只基金，但您自己研究后觉得一般，您会怎么做？"

场景5：过度自信测试
"您觉得您的投资水平在普通投资者中处于什么位置？前10%？前30%？平均水平？后30%？"
```
