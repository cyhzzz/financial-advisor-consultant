# Skill: 基金筛选与诊断 (Fund Screening & Diagnosis)

## Description
基于客户风险偏好和投资目标，智能筛选基金产品并进行持仓诊断。适配中国公募基金市场，支持 Wind、同花顺等数据源。

## Trigger Conditions
- 用户需要筛选符合条件的基金
- 客户询问"有哪些适合我的基金"
- 需要诊断现有持仓组合
- 需要对比多只基金产品

## Workflow

### 步骤 1: 明确筛选需求
通过提问获取关键信息：
```
必问项:
- 风险承受能力 (R1-R5)
- 投资期限 (短期<1年/中期1-3年/长期>3年)
- 预期收益目标

选问项:
- 基金类型偏好 (货币型/债券型/混合型/股票型/指数型/QDII)
- 规模要求 (如 >10亿)
- 成立年限 (如 >3年)
- 费率敏感度
```

### 步骤 2: 设定筛选条件
基于中国公募基金市场常用指标：

**定量指标**:
| 维度 | 指标 | 筛选阈值参考 |
|------|------|-------------|
| 收益 | 近1年/3年/5年收益率 | 同类前1/3 |
| 风险 | 年化波动率 | 低于同类平均 |
| 风险调整 | 夏普比率 | >0.5 |
| 回撤 | 最大回撤 | 同类前1/2 |
| 规模 | 最新规模 | >5亿（债基>10亿） |
| 费率 | 管理费+托管费 | 同类偏低 |
| 稳定性 | 基金经理任职年限 | >2年 |

**定性指标**:
- 基金公司投研实力（非货规模、获奖情况）
- 基金经理投资理念一致性
- 历史风格漂移情况

### 步骤 3: 执行筛选
```
数据来源优先级:
1. Wind 基金数据库 (首选)
2. 同花顺 iFinD
3. 恒生聚源
4. 基金业协会数据
5. 第三方平台 (好买基金、天天基金)
```

### 步骤 5: 持仓诊断流程
针对已有持仓的客户：

```
诊断维度:
1. 组合层面
   - 整体风险收益特征
   - 资产配置偏离度
   - 行业/风格集中度
   - 费率侵蚀分析

2. 单品层面
   - 各基金业绩归因
   - 风格漂移检测
   - 基金经理稳定性
   - 同类排名变化

3. 问题识别
   - 过度集中于单一资产/行业
   - 持有过多同类基金
   - 高费率产品占比过高
   - 业绩持续落后同类

4. 优化建议
   - 调仓方案（调出/调入/比例调整）
   - 替代产品推荐
   - 调仓执行计划
```

## Examples

### Example 1: 基金筛选
**User**: "帮我筛选几只稳健型的债券基金，投资期限 2 年左右"

**执行过程**:
1. 确认需求：R2 稳健型，中期投资，纯债/一级债基
2. 设定条件：近3年年化>4%，最大回撤<3%，规模>10亿
3. 执行筛选：从 2000+ 债基中筛选出 30 只
4. 排序打分：夏普比率、卡玛比率、基金经理稳定性
5. 生成报告：Top 5 推荐，附详细对比表

**输出**: [基金筛选报告 - 稳健型债券基金]

### Example 2: 持仓诊断
**User**: "帮我诊断一下客户张先生的持仓组合，看看有什么问题"

**执行过程**:
1. 导入持仓数据（5 只基金，总市值 150 万）
2. 组合层面分析：权益占比 75%，超出客户 R3 风险偏好
3. 单品诊断：发现 2 只基金业绩持续落后，1 只风格漂移
4. 问题汇总：集中度风险（高）、单品问题（中）、费率侵蚀（低）
5. 优化方案：提供保守/平衡/积极三档调整建议

**输出**: [投资组合诊断报告 - 张先生]
