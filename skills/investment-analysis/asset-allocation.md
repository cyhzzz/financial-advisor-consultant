# Skill: 资产配置建模 (Asset Allocation Modeling)

## Description
根据客户风险偏好和投资目标，构建个性化的资产配置方案。支持战略资产配置(SAA)和战术资产配置(TAA)。

## Trigger Conditions
- 新客户开户需要制定资产配置方案
- 定期年度/半年度资产配置再平衡
- 市场环境重大变化需要调整配置
- 客户人生阶段变化（如结婚、生子、退休）

## Workflow

### 步骤 1: 客户需求采集

**核心问题清单**:
```
1. 投资目标
   - 资金用途: [ ] 养老 [ ] 子女教育 [ ] 购房 [ ] 财富增值 [ ] 其他___
   - 目标金额: ______ 万元
   - 目标时间: ______ 年后

2. 风险承受能力
   - 年龄: ______ 岁
   - 投资经验: [ ] 无 [ ] <1年 [ ] 1-3年 [ ] 3-5年 [ ] >5年
   - 最大可接受亏损: [ ] 5%以内 [ ] 10%以内 [ ] 20%以内 [ ] 30%以上

3. 流动性需求
   - 预计赎回频率: [ ] 随时可能 [ ] 半年内不会 [ ] 1年内不会
   - 最低保留金额: ______ 万元
```

### 步骤 2: 风险测评与画像

**风险等级判定矩阵**:

| 风险等级 | 年龄范围 | 投资经验 | 可接受最大回撤 | 适合配置 |
|----------|----------|----------|----------------|----------|
| R1 保守型 | >55岁 | <1年 | <5% | 货币/短债 ≥80% |
| R2 稳健型 | 45-55岁 | 1-3年 | 5-10% | 债券 ≥60%, 权益 ≤40% |
| R3 平衡型 | 35-45岁 | 3-5年 | 10-20% | 股债均衡 40-60% |
| R4 成长型 | 25-35岁 | >5年 | 20-30% | 权益 ≥60%, 债券 ≤40% |
| R5 进取型 | <35岁 | >5年 | >30% | 权益 ≥80%, 含另类资产 |

### 步骤 3: 战略资产配置(SAA)模型

**模型选择**:

```
模型1: 均值-方差优化 (Mean-Variance Optimization)
- 适用: 数据充足、资产类别相关性稳定
- 输入: 各资产历史收益率、波动率、相关系数矩阵
- 输出: 有效前沿 + 最优配置比例

模型2: 风险平价 (Risk Parity)
- 适用: 希望各资产对组合风险贡献均衡
- 逻辑: 波动率低的资产配更高权重，使风险贡献相等
- 输出: 风险均衡配置权重

模型3: 目标日期策略 (Glide Path)
- 适用: 养老/教育等目标日期型投资
- 逻辑: 随目标日期临近，自动降低权益比例
- 输出: 动态调整的配置路径

模型4: 美林时钟/全天候策略
- 适用: 宏观视角的大类资产配置
- 逻辑: 根据经济周期/通胀环境调整股债商比例
- 输出: 周期适应性配置
```

**资产类别定义（中国版）**:

| 资产类别 | 细分类别 | 代表产品/指数 | 预期收益 | 波动率 |
|----------|----------|---------------|----------|--------|
| 现金管理 | 货币基金、短债基金 | 余额宝、短融债基 | 2-3% | <1% |
| 固收类 | 纯债基金、一级债基 | 中债总财富指数 | 3-5% | 2-4% |
| 固收+ | 二级债基、偏债混合 | 偏债混合基金指数 | 5-8% | 5-8% |
| 权益类 | 偏股混合、股票基金 | 沪深300、中证500 | 8-12% | 15-25% |
| 指数类 | 宽基ETF、行业ETF | 科创50、行业指数 | 同权益 | 同权益 |
| QDII | 海外股票、债券基金 | 纳斯达克100、美债 | 8-15% | 15-25% |
| 另类 | REITs、商品基金 | 黄金ETF、REITs | 5-10% | 10-20% |

### 步骤 4: 战术资产配置(TAA)调整

**TAA 触发条件**:
```
定期调整:
- 每季度末进行再平衡检查
- 每年度重新评估 SAA

事件驱动:
- 宏观经济重大变化（央行政策转向、通胀超预期）
- 市场极端情绪（恐慌/贪婪指数极值）
- 客户人生阶段变化（结婚、生子、退休）
- 单一资产大幅上涨/下跌导致配置偏离
```

**TAA 调整幅度限制**:
```
- 单资产类别偏离 SAA 不超过 ±10%
- 权益类整体仓位偏离不超过 ±15%
- 调整频率不超过每季度一次（除极端情况）
```

### 步骤 5: 压力测试与情景分析

**压力测试场景（中国版）**:

| 情景 | 描述 | 权益冲击 | 固收冲击 | 组合回撤 |
|------|------|----------|----------|----------|
| 2018 贸易战 | 中美贸易摩擦升级 | -25% | +2% | -15% |
| 2020 疫情冲击 | 新冠疫情爆发 | -15% | +1% | -10% |
| 2022 俄乌冲突 | 地缘政治危机 | -20% | +0% | -12% |
| 加息周期 | 央行持续加息 | -15% | -5% | -12% |
| 流动性危机 | 信用违约潮 | -30% | -3% | -20% |

**输出要求**:
- 组合在各情景下的预期回撤
- 最坏情况下的亏损金额（绝对值）
- 客户心理承受度匹配度评估

### 步骤 6: 生成配置方案文档

**完整方案包含**:
1. 客户画像与风险评估
2. 战略资产配置(SAA)方案
3. 具体基金产品推荐清单
4. 战术调整规则
5. 再平衡计划
6. 风险提示与压力测试结果

### 2.4 战术配置(TAA)规则

**定期调整**:
- 每季度末检查配置偏离度
- 单一资产偏离目标 ±5% 触发再平衡

**事件驱动调整**:
- 宏观经济重大变化（如央行政策转向）
- 单一资产估值极端（如股债性价比<1.5 或 >3.0）
- 客户人生阶段变化

**调整限制**:
- 单次调整幅度不超过 ±10%
- 年度调整不超过 3 次（极端情况除外）

## 关键指标监控

| 监控指标 | 目标值 | 警戒线 | 行动阈值 |
|----------|--------|--------|----------|
| 组合波动率 | 8-12% | >15% | 减仓权益 |
| 最大回撤 | <-12% | <-15% | 启动保护 |
| 夏普比率 | >0.6 | <0.4 | 优化结构 |
| 偏离度检查 | ±5% | ±8% | 触发再平衡 |
