# Skill: 组合诊断分析 (Portfolio Diagnosis)

## Description
对客户现有投资组合进行全面诊断，识别风险暴露、业绩归因、风格漂移等问题，并提供优化建议。

## Trigger Conditions
- 定期季度/年度组合回顾
- 客户主动要求诊断
- 市场大幅波动后
- 重大调仓前

## Workflow

### 步骤 1: 数据获取与清洗

**采集数据**:
- 当前持仓明细（基金代码、份额、市值、成本）
- 历史交易记录（申赎记录、分红记录）
- 客户现金流（追加投资/赎回记录）

**数据清洗**:
- 核对持仓市值与总资产的匹配性
- 处理停牌基金、QDII 净值滞后
- 统一时间基准（建议用 T-1 日净值）

### 步骤 2: 组合层面分析

**收益分析**:
```
□ 累计收益率（绝对收益）
□ 年化收益率（时间加权）
□ 资金加权收益率（IRR，考虑现金流）
□ 滚动收益（近1月/3月/6月/1年/2年/3年）
□ 月度/季度胜率
```

**风险分析**:
```
□ 年化波动率（标准差）
□ 最大回撤（回撤幅度、持续时间、恢复时间）
□ 下行波动率（半方差）
□ VaR/CVaR（在险价值）
□ 尾部风险（峰度、偏度）
```

**风险调整后收益**:
```
□ 夏普比率 (Sharpe Ratio) = (Rp - Rf) / σp
□ 索提诺比率 (Sortino Ratio) = (Rp - Rf) / σd
□ 卡玛比率 (Calmar Ratio) = Rp / |最大回撤|
□ 信息比率 (Information Ratio) = α / σ(α)
□ 特雷诺比率 (Treynor Ratio) = (Rp - Rf) / β
```

**资产配置分析**:
```
□ 大类资产配置比例（权益/固收/现金/另类）
□ 与目标配置的偏离度
□ 配置集中度（HHI 指数）
□ 行业配置分布
□ 风格配置（价值/成长/大盘/中小盘）
□ 地域配置（A股/港股/海外）
```

**相关性分析**:
```
□ 各基金之间的相关性矩阵
□ 组合与基准的相关性 (R²)
□ Beta 系数
□ 组合分散度评估
```

### 步骤 3: 单品层面分析

**业绩归因 (Brinson 模型)**:
```
超额收益 = 资产配置效应 + 个股选择效应 + 交互效应

资产配置效应 = Σ(Wpi - Wbi) × Rbi
个股选择效应 = ΣWbi × (Rpi - Rbi)
交互效应 = Σ(Wpi - Wbi) × (Rpi - Rbi)

其中:
Wpi = 组合中第i类资产的权重
Wbi = 基准中第i类资产的权重
Rpi = 组合中第i类资产的收益
Rbi = 基准中第i类资产的收益
```

**风格漂移检测**:
```
□ 历史风格暴露追踪（Size/Value/Momentum/Quality等）
□ 风格稳定性评分
□ 风格漂移预警
□ 与基金合同约定的投资风格一致性检查
```

**持仓分析**:
```
□ 前十大重仓集中度
□ 换手率分析
□ 持仓久期/持有周期
□ 新增/退出持仓
□ 加仓/减仓操作
```

**基金经理评估**:
```
□ 任职期间业绩
□ 管理规模变化
□ 投资理念一致性
□ 团队稳定性
□ 获奖情况
```

### 步骤 4: 问题识别与诊断

**组合层面问题**:
```
□ 资产配置偏离
  - 权益占比过高/过低
  - 行业/风格过度集中
  - 地域配置不均衡

□ 风险暴露异常
  - 波动率过高
  - 最大回撤超限
  - 尾部风险积聚

□ 收益表现不佳
  - 跑输基准
  - 跑输同类平均
  - 风险调整后收益偏低

□ 组合效率问题
  - 相关性过高，分散不足
  - 费率侵蚀严重
  - 申赎成本高
```

**单品层面问题**:
```
□ 业绩问题
  - 长期业绩落后同类
  - 近期业绩大幅下滑
  - 业绩持续性差

□ 风格漂移
  - 实际投资风格与合同不符
  - 风格频繁切换

□ 管理问题
  - 基金经理变更
  - 规模暴增/暴跌
  - 公司/团队动荡

□ 风险问题
  - 持仓过度集中
  - 杠杆率过高
  - 流动性风险
```

**行为层面问题**:
```
□ 择时错误
  - 高点买入，低点卖出
  - 频繁申赎，追涨杀跌

□ 配置错误
  - 权益仓位与风险承受能力不匹配
  - 过度集中或过度分散

□ 持有期过短
  - 没有给予策略足够的时间验证
  - 被短期波动影响
```

### 注意事项
⚠️ 调仓风险:
- 市场短期波动风险
- 申赎费用成本（持有期不足可能产生高额赎回费）
- 调仓期间的现金踏空风险

💡 建议:
- 分批调仓，避免择时风险
- 关注基金申赎状态（暂停申赎、大额限购等）
- 保留适当现金缓冲

### 定期检查
- **每周**: 市场回顾，必要时调整 TAA
- **每月**: 组合回顾，单品业绩监控
- **每季**: 深度诊断，SAA 再平衡检查
- **每年**: 客户风险评估更新，长期目标回顾

### 触发再平衡的条件
1. 单一资产类别偏离目标配置 ±5%
2. 单一基金占比超过组合的 20%
3. 客户风险承受能力发生重大变化
4. 市场环境发生结构性变化

## 九、附录

### 分析工具与方法
- **Brinson 归因**: 资产配置 vs 个股选择贡献
- **Barra 多因子模型**: 风格暴露分析
- **Campisi 归因**（固收）: 收入效应、国债效应、利差效应、择券效应
- **TM/HM 模型**: 择时能力评估

### 参考基准
- 权益类: 沪深300指数、中证500指数、偏股混合基金指数
- 固收类: 中债总财富指数、中短债基金指数
- 综合: 万得全A、偏股/偏债基金指数
