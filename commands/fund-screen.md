# Command: /fund-screen

## Description
根据客户风险偏好和投资目标，智能筛选符合条件的基金产品。

## Usage
```
/fund-screen [options]

Options:
  --risk, -r          风险等级 (R1/R2/R3/R4/R5)
  --type, -t          基金类型 (stock/bond/mixed/index/qdii/money)
  --term, -T          投资期限 (short/medium/long)
  --size, -s          规模要求 (min:XX亿)
  --fee, -f           费率上限 (max:X.X%)
  --manager, -m       基金经理任职年限 (min:X年)
  --top, -n           返回数量 (default:5)
  --output, -o        输出格式 (table/report)
```

## Examples

### 示例 1: 为稳健型客户筛选债券基金
```
/fund-screen --risk R2 --type bond --term medium --size 10 --top 5
```

## Implementation Notes

### 筛选算法
1. **初筛**: 根据硬性条件（风险等级、类型、规模、费率）过滤
2. **排序**: 根据多因子模型打分（收益、风险调整后收益、稳定性）
3. **精选**: 结合定性因素（基金经理、公司实力）人工复核

### 数据源
- Wind 基金数据库
- 同花顺 iFinD
- 好买基金
- 天天基金

### 更新频率
- 日度: 净值、规模数据
- 季度: 持仓、业绩归因
- 年度: 完整年度业绩
