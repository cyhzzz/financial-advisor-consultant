# Command: /market-note

## Description
快速生成市场点评内容，支持多种风格、长度和用途。

## Usage
```
/market-note [options]

Options:
  --type, -t          类型 (morning/evening/weekly/event)
  --style, -s         风格 (professional/popular/brief/humorous)
  --length, -l        长度 (short/medium/long)
  --focus, -f         聚焦板块/主题
  --audience, -a      受众 (beginner/intermediate/advanced)
  --tone, -T          情绪基调 (optimistic/neutral/cautious)
  --output, -o        输出 (text/wechat/ppt)
```

## Examples

### 示例 1: 早盘点评（通俗风格，适合朋友圈）
```
/market-note --type morning --style popular --length short --audience beginner
```

**输出**:
```
📊 {日期} 早盘提示

各位早上好！🌞

昨夜美股{涨跌}，{主要原因}。今天A股大概率{高开/低开/平开}。

💡 今日关注:
• {关注事项1}
• {关注事项2}
• {关注事项3}

操作建议: {建议}

⚠️ 提醒: {风险提示}

祝投资顺利！💰
```

```
- 转场: 简洁流畅
- 封面: 醒目标题 + 人物/数据图表

【发布建议】
- 发布时间: 工作日 19:00-21:00 或 午间 12:00-13:00
- 标签: #AI #人工智能 #投资 #基金 #理财
- 文案: 配合视频内容，提炼 3 个核心观点
```

## Quality Checklist

**内容发布前检查**:
- [ ] 事实数据准确无误

- [ ] 合规风险已排查（无承诺收益、无虚假宣传）

- [ ] 适合目标受众阅读水平
